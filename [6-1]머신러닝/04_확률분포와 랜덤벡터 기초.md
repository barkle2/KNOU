# 확률분포와 랜덤벡터 기초

--------------------------------------------------------------------------------


요약 (Executive Summary)

* 확률의 정의와 공리: 확률은 고전적, 빈도적, 주관적 관점에서 정의될 수 있으나, 수학적으로는 비음수성, 전체 확률의 합(1), 배반 사건의 가법성이라는 세 가지 공리를 만족해야 한다.
* 랜덤 변수(확률변수): 랜덤 실험의 결과를 수치로 대응시키는 함수로, 이산 랜덤 변수와 연속 랜덤 변수로 구분된다. 이산형은 확률 질량 함수(PMF)를, 연속형은 확률 밀도 함수(PDF)를 사용하여 확률 구조를 설명한다.
* 주요 확률분포: 균등 분포(이산/연속), 이항 분포, 정규 분포가 대표적이다. 특히 정규 분포는 여러 독립적 요소가 누적될 때 나타나는 핵심적인 분포이다.
* 컴퓨터 구현: Python의 scipy 패키지를 활용하여 실현값 추출(RVS), 분위수 계산(PPF), 누적 확률(CDF), 확률 밀도/질량(PDF/PMF) 값을 계산할 수 있다.
* 랜덤 벡터: 두 개 이상의 랜덤 변수를 동시에 고려하는 개념으로, 결합 확률 분포, 주변 분포, 조건부 분포를 통해 변수 간의 관계와 개별적 특성을 분석한다.

## 1. 확률의 기초 개념 및 공리

### 1.1 랜덤 실험과 관련 용어

* 랜덤 실험(Random Experiment): 어떤 결과가 나올지 사전에 알 수 없으나, 동일 조건에서 반복 가능하며 모든 가능한 결과의 집합을 알고 있는 실험(예: 동전 던지기, 주사위 굴리기).
* 표본 공간(Sample Space, S): 랜덤 실험에서 발생 가능한 모든 결과의 집합.
* 사건(Event, A): 표본 공간의 부분 집합으로, 특정 관심 대상을 의미한다.

### 1.2 확률의 세 가지 정의

|정의|유형|설명|비고|
|:-:|:-:|:-:|:-:|
|고전적 확률|모든 결과가 동일한 가능성을 가질 때, (관심 경우의 수 / 전체 경우의 수)로 정의|주사위 눈금 1이 나올 확률 (1/6)||
|빈도적 확률|실험을 무한히 반복할 때 사건이 발생하는 비율로 정의|불공정한 주사위의 확률 추정에 유용||
|주관적 확률|개인의 신념이나 정보에 기반하여 확률을 부여|베이지안 확률의 사전 분포 개념과 연관||

### 1.3 확률의 3대 공리

모든 확률 정의는 다음의 공리를 만족해야 한다.

1. 비음수성: 모든 사건 A에 대해 P(A) \geq 0이다.
2. 규격화: 표본 공간 전체의 확률은 1이다 (P(S) = 1).
3. 가법성: 서로 동시에 일어날 수 없는 배반 사건들의 합집합 확률은 각 사건 확률의 합과 같다.

## 2. 랜덤 변수와 확률 분포

### 2.1 랜덤 변수(Random Variable)의 정의

랜덤 실험의 결과를 수치로 대응시켜 주는 함수이다. 실현값(소문자 x)과 랜덤 변수 자체(대문자 X)를 구분하여 표기한다.

* 결정적 요소: 모델에 의해 설명되는 인과관계나 모수(Signal).
* 확률적 요소: 모델이 설명하지 못하는 불확실성(Noise, Error).

### 2.2 이산형 vs 연속형 랜덤 변수

|구분|이산 랜덤 변수 (Discrete)|연속 랜덤 변수 (Continuous)|
|:-:|:-:|:-:|
|특징|결과값을 셀 수 있음 (Countable)|실수 구간 내 임의의 값을 가짐 (Uncountable)|
|함수|확률 질량 함수 (PMF)|확률 밀도 함수 (PDF)|
|확률 정의|특정 값에서의 확률이 직접 정의됨|특정 값의 확률은 0이며, 구간 면적으로 정의|
|대표 예시|주사위 눈금, 성공 횟수|체온, 키, 몸무게|

### 2.3 누적 분포 함수 (CDF)

$F(x) = P(X \leq x)$로 정의되며, 특정 값 x보다 작거나 같을 확률을 나타낸다. 이산형의 경우 계단식 함수 형태를 띠며, 연속형의 경우 연속적인 곡선 형태를 가진다.

## 3. 주요 확률분포의 특성

### 3.1 이산 균등 분포 (Discrete Uniform Distribution)

모든 가능한 값에 대해 동일한 확률을 할당한다. 주사위 던지기가 대표적이다.

* PMF: $1 / (b - a + 1)$
* 평균: $(a + b) / 2$

### 3.2 이항 분포 (Binomial Distribution)

독립적인 베르누이 시행(결과가 0 또는 1)을 n번 반복했을 때 성공 횟수의 분포이다.

* 모수: n(시행 횟수), p(성공 확률)
* 평균: np
* 분산: np(1-p)

### 3.3 연속 균등 분포 (Continuous Uniform Distribution)

특정 구간 [a, b] 내의 모든 값에 대해 균일한 확률 밀도를 가진다.

* PDF: $1 / (b - a)$ (단, $a \leq x \leq b$)

### 3.4 정규 분포 (Normal Distribution)

자연계의 많은 현상을 설명하는 종 모양의 대칭 분포로, 중심 극한 정리에 의해 독립적인 요소들이 많이 누적될 때 나타난다.

* 모수: $\mu$(평균), $\sigma^2$(분산)
* 특징: 표준 정규 분포는 $\mu=0, \sigma=1$인 경우를 말한다.

## 4. Python 프로그래밍 메서드 활용

확률적 분석을 위해 scipy.stats 모듈의 객체 지향 프로그래밍(OOP) 개념을 사용한다.

* RVS (Random Variates): 정의된 분포로부터 실현값을 무작위로 추출한다.
* PPF (Percent Point Function): 분위수 값을 계산한다. 누적 확률(입력)을 통해 해당 x값(출력)을 찾는다. (예: 하위 2.5%에 해당하는 x값은 -1.96)
* CDF (Cumulative Distribution Function): x값(입력)을 통해 누적 확률(출력)을 계산한다.
* PDF/PMF: 특정 x값에서의 밀도(연속형) 또는 확률 질량(이산형)을 계산한다.

## 5. 랜덤 벡터 (Random Vector)

### 5.1 정의 및 구조

두 개 이상의 랜덤 변수를 구성 요소로 가지는 벡터이다. 일반적으로 열 벡터(Column Vector) 형식을 기본으로 한다.

* 평균 벡터: 각 랜덤 변수의 평균들로 구성된 벡터.
* 분산-공분산 행렬: 대각 원소에는 개별 변수의 분산이, 비대각 원소에는 변수 간의 공분산이 위치하는 정사각 행렬이다.

### 5.2 확률 분포의 구분

랜덤 벡터 내의 변수들 사이의 관계를 설명하기 위해 세 가지 분포 개념을 사용한다.

|분포 유형|설명|
|:-:|:-:|
|결합 확률 분포 (Joint)|벡터 내 모든 랜덤 변수의 발생 가능성을 동시에 고려한 분포.|
|주변 분포 (Marginal)|다른 변수를 무시하고(전체 고려하여 합산/적분) 특정 변수만의 분포를 추출.|
|조건부 분포 (Conditional)|특정 변수가 어떤 값으로 고정되었을 때, 나머지 변수의 분포를 정의.|

이러한 다변량 접근 방식은 단일 변수 분석을 넘어 변수 간의 상호 의존성을 파악하는 데 필수적인 도구로 활용된다.
