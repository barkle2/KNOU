# 선형 함수 근사

--------------------------------------------------------------------------------


핵심 요약 (Executive Summary)

* 모델의 확장성: 단순 선형 모델의 한계를 극복하기 위해 **기저 함수(Basis Function)**를 도입, 입력값의 선형 결합을 통해 복잡한 비선형 데이터를 모델링하는 선형 기저 함수 모델을 구축한다.
* 최적 파라미터 추정: 오차 제곱합을 최소화하는 **최소 자승법(Least Squares)**은 정규분포 노이즈 가정 하에서 **최대 우도 추정(MLE)**과 동일한 결과를 도출하며, 결정론적 방법과 확률론적 방법의 연결 고리를 제공한다.
* 과적합 방지: 모델이 훈련 데이터에 지나치게 치우치는 문제를 해결하기 위해 파라미터 크기에 페널티를 부여하는 정규화(Regularization) 기법을 적용하여 모델의 일반화 성능을 향상시킨다.
* 베이지안 접근의 이점: 파라미터를 고정된 값이 아닌 확률 변수로 취급하는 베이지안 선형 회귀는 데이터가 축적됨에 따라 사후 분포를 갱신하며, 단순 점 예측을 넘어 **예측 구간(Predictive Interval)**과 같은 불확실성을 정량적으로 제시할 수 있다.


--------------------------------------------------------------------------------


## 1. 선형 기저 함수 모델 (Linear Basis Function Models)

### 1.1 정의 및 구조

단순 선형 회귀($y = w_0 + w_1x$)는 현실의 복잡한 비선형 문제를 해결하는 데 한계가 있다. 이를 보완하기 위해 입력 변수 x에 기저 함수 $\phi(x)$를 적용하여 다음과 같이 표현한다.

* 수식: $$y(x, w) = w^T \phi(x)$$
* 구성 요소:
  * 바이어스($w_0$): 입력값에 영향을 받지 않는 상수로, $\phi_0(x) = 1$로 설정된다.
  * 웨이트($w_1 \dots w_{M-1}$): 각 기저 함수의 가중치이다.
  * 파라미터 벡터(w): 바이어스와 웨이트를 포함한 M개의 파라미터 집합이다.

### 1.2 기저 함수의 종류

모델의 표현력은 선택하는 기저 함수의 종류에 따라 결정된다.

* 다항식 기저 (Polynomial Basis): 입력값의 거듭제곱을 사용하여 비선형성을 확보한다.
* RBF 기저 (Radial Basis Function): 중심점으로부터의 거리에 기반한 함수를 사용한다.
* 푸리에 기저 (Fourier Basis): 주기적인 형태를 모델링할 때 유리하다.

### 1.3 디자인 매트릭스 (Design Matrix, $\Phi$)

데이터셋의 모든 기저 함수 값을 행렬 형태로 모은 것을 디자인 매트릭스라 한다. 데이터 수가 N, 피처 수가 M일 때 $N \times M$ 크기를 가지며, 다항식 기저를 사용할 경우 반데르몽드(Vandermonde) 행렬 형태를 띤다.


--------------------------------------------------------------------------------

## 2. 파라미터 추정 방법론

### 2.1 최소 자승법 (Least Squares, LS)

실제 관측값 y와 모델 예측값 사이의 **오차 제곱합(SSE)**을 최소화하는 파라미터 w를 구하는 방법이다.

* 최소 자승해 공식: $w = (\Phi^T \Phi)^{-1} \Phi^T y$
* 이 공식은 미분값이 0이 되는 지점을 찾는 과정에서 유도되며, 이를 **정규 방정식(Normal Equation)**이라 부른다.

### 2.2 최대 우도 추정 (Maximum Likelihood Estimation, MLE)

관측 데이터에 정규분포 노이즈가 포함되어 있다고 가정할 때, 관측치가 나타날 개연성(우도)을 최대화하는 파라미터를 찾는 방식이다.

* 정규분포 노이즈를 가정할 경우, **로그 라이클리우드(Log-Likelihood)**를 최대화하는 해는 최소 자승법의 해와 수학적으로 동일하다.
* 즉, 데이터에 가장 잘 맞는 파라미터를 찾는 결정론적 방식이 확률론적 해석으로 뒷받침됨을 의미한다.


--------------------------------------------------------------------------------


## 3. 정규화와 일반화 (Regularization)

### 3.1 과적합(Overfitting) 문제

모델이 훈련 데이터의 노이즈까지 학습하여 파라미터 w 값이 비정상적으로 커지고, 새로운 데이터에 대한 예측 성능이 저하되는 현상이다.

### 3.2 정규화된 최소 자승법 (Regularized Least Squares)

손실 함수에 파라미터 크기에 비례하는 페널티 항을 추가하여 모델의 복잡도를 억제한다.

* 정규화 공식: $w = (\Phi^T \Phi + \lambda I)^{-1} \Phi^T t$
* 주요 특징:
  * $\lambda$: 정규화 강도를 조절하는 파라미터이다. $\lambda$가 크면 파라미터가 0에 가까워져 모델이 단순해진다.
  * 수치적 안정성: $\Phi^T \Phi$가 역행렬이 존재하지 않는 경우(Singular Matrix)에도 $\lambda I$를 더함으로써 안정적인 역행렬 계산이 가능해진다.

## 4. 베이지안 선형 회귀 (Bayesian Linear Regression)

### 4.1 베이지안 관점의 핵심

베이지안 방식은 파라미터 w를 고정된 값이 아닌 확률 변수로 간주한다. 데이터를 관찰하기 전의 믿음인 **사전 분포(Prior)**에 데이터의 우도를 반영하여 **사후 분포(Posterior)**를 도출한다.

| 단계 | 설명 |
|:---:|:---:|
| 사전 분포 (Prior) | 데이터 학습 전 w에 대한 분포. 보통 분산이 큰 정규분포로 설정한다. |
| 우도 (Likelihood) | 주어진 w에 대해 현재 데이터가 관측될 확률. |
| 사후 분포 (Posterior) | 사전 분포와 우도를 곱하여 갱신된 분포. 데이터가 많아질수록 뾰족한 형태가 된다. |

### 4.2 예측 분포 (Predictive Distribution)

베이지안 모델의 가장 큰 장점은 새로운 입력 $x^*$에 대해 단일값이 아닌 확률 분포 $p(t^* | x^*, D)$를 제공한다는 것이다.

* 예측 평균: 모델이 예측하는 가장 가능성 높은 지점.
* 예측 구간: 예측값의 불확실성을 나타내며, 시각화 시 평균선을 중심으로 범위를 형성한다.
* 하이퍼 파라미터: 사전 분포의 프리시전($\alpha$)과 노이즈의 프리시전($\beta$)을 사용하여 불확실성을 제어한다.


--------------------------------------------------------------------------------


## 5. 결론

본 분석을 통해 선형 기저 함수 모델은 단순한 회귀를 넘어 다양한 비선형 데이터에 유연하게 대응할 수 있는 강력한 도구임을 확인하였다. 특히 최소 자승법에서 시작하여 정규화를 통한 오버피팅 완화, 그리고 베이지안 접근을 통한 불확실성 예측으로 이어지는 흐름은 머신러닝의 파라미터 추정 및 모델 평가에 있어 핵심적인 이론적 토대를 제공한다. 딥러닝과 달리 예측의 근거가 되는 불확실성(예측 구간)을 제시할 수 있는 베이지안 모델의 특성은 높은 신뢰성이 요구되는 데이터 분석 환경에서 중요한 가치를 지닌다.
