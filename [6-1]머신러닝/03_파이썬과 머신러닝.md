# 제3강 파이썬과 머신러닝

본 문서는 파이썬을 활용한 머신러닝의 기초가 되는 모듈 및 라이브러리 체계와 데이터 분석의 핵심 3대 라이브러리인 **판다스(Pandas), 넘파이(NumPy), 맷플롯립(Matplotlib)**의 활용법을 상술한다.

주요 분석 대상인 '붓꽃(Iris) 데이터셋'을 통해 데이터 로드, 전처리, 수치 분석, 그리고 시각화에 이르는 일련의 과정을 다룬다. 특히 데이터의 규모를 일정하게 맞추는 **스케일링(Standardization, Min-Max Scaling)**과 고차원 데이터의 핵심 성분을 추출하는 **주성분 분석(PCA)**의 수치적 구현 방법을 포함한다. 데이터 분석의 목적은 라이브러리를 통해 원시 데이터를 정제하고 시각화하여, 인간이 데이터 내의 패턴(예: 종별 구분 기준)을 직관적으로 파악할 수 있도록 돕는 데 있다.

## 1. 파이썬 모듈 및 라이브러리 체계

### 1.1 모듈(Module)과 라이브러리(Library)의 정의

* 모듈: 함수, 클래스, 변수 등을 저장한 .py 확장자 파일로, 조립을 위한 규격화된 부품 역할을 수행한다.
* 라이브러리: 특정 기능을 수행하는 모듈들의 집합이다. 파이썬 내장 '표준 라이브러리'와 별도 설치가 필요한 '서드 파티(3rd Party) 라이브러리'(Pandas, NumPy 등)로 구분된다.

1.2 모듈 호출 방식 및 주의사항

|호출 방식|특징|비고|
|:-:|:-:|:-:|
|import 모듈명|모듈 내 기능을 모듈명.기능 형태로 사용|가장 기본적이고 안전함|
|import 모듈명 as 별명|긴 모듈명을 짧은 별명으로 대체 (예: import pandas as pd)|실무에서 가장 선호됨|
|from 모듈명 import 기능|특정 기능만 직접 호출하여 사용|모듈명 생략 가능|
|from 모듈명 import *|모듈 내 모든 기능을 호출(와일드카드)|권장되지 않음: 기존 변수명과의 충돌 위험 상존|

## 2. 판다스(Pandas)를 활용한 데이터 처리 및 필터링

판다스는 '패널 데이터(Panel Data)'에서 유래하였으며, CSV 등 외부 데이터 파일을 읽고 표 형태의 데이터 프레임(Data Frame) 객체로 처리하는 데 특화되어 있다.

### 2.1 데이터 파악 및 요약 메서드

* read_csv(): CSV 파일을 데이터 프레임으로 로드한다.
* head(): 데이터의 상위 5개 행을 출력하여 데이터 구조를 맛보기로 확인한다.
* info(): 전체 행/열 개수, 컬럼명, 데이터 타입(float64, object 등), 결측치 유무를 확인한다.
* describe(): 평균, 표준편차, 사분위수, 최솟값/최댓값 등 기술 통계량을 한눈에 보여준다.

### 2.2 인덱싱(Indexing) 및 조건 필터링

* 레이블 기반 인덱싱: 딕셔너리 스타일(df['컬럼명'])로 특정 열을 추출한다. 추출 후 value_counts()를 통해 범주별 데이터 개수를 파악할 수 있다.
* 위치 기반 인덱싱(iloc): 정수 인덱스를 사용하여 행과 열의 위치로 데이터를 추출한다. (예: iloc[0, 0]은 1행 1열 값)
* 조건 필터링: 불리언(Boolean) 조건을 사용하여 특정 기준을 만족하는 행만 추출한다.
  * 단일 조건: df[df['species'] == 'setosa']
  * 다중 조건: &(AND) 또는 |(OR) 연산자를 사용하여 복합적인 조건을 적용한다. (예: 꽃받침 길이가 6.5 이상이면서 종이 'versicolor'인 데이터 추출)

## 3. 넘파이(NumPy)를 활용한 수치 분석

넘파이는 다차원 배열(Array) 연산에 최적화된 라이브러리로, 대규모 수치 계산 및 시뮬레이션에서 판다스보다 빠른 성능을 제공한다.

### 3.1 주요 특징 및 통계 계산

* 데이터 변환: 판다스 데이터 프레임에 .values를 적용하여 넘파이 배열로 변환할 수 있다.
* 축(Axis) 설정: axis=0은 열(Column) 방향 연산을, axis=1은 행(Row) 방향 연산을 의미한다. 평균(mean), 표준편차(std) 계산 시 필수적으로 고려해야 한다.
* 상관관계 분석: np.corrcoef() 함수를 사용하여 변수 간의 상관계수 행렬을 산출한다. 상관계수는 -1에서 1 사이의 값을 가지며, 변수 간 패턴의 유사성을 나타낸다.

### 3.2 데이터 전처리(Scaling) 기법

모델의 학습 효율을 높이기 위해 서로 다른 단위의 변수들을 일정 범위로 조정하는 과정이다.

1. 표준화(Standardization):
  * 데이터의 평균을 0, 분산을 1로 변환한다.
  * 공식: (데이터 - 평균) / 표준편차
  * 결과물에서 평균이 매우 작은 수(예: 10^-16)로 나타나는 것은 부동 소수점 오차에 의한 것이며, 실질적으로 0과 다름없음을 의미한다.
2. 민맥스 스케일링(Min-Max Scaling):
  * 데이터의 최솟값을 0, 최댓값을 1로 변환하여 0~1 사이의 범위를 갖게 한다.
  * 공식: (데이터 - 최솟값) / (최댓값 - 최솟값)

## 4. 주성분 분석(PCA)의 이해 및 구현

PCA는 데이터의 정보를 최대한 유지하면서 차원(변수의 개수)을 축소하는 기법이다. 넘파이의 선형 대수 모듈(linalg)을 통해 구현 가능하다.

* 구현 절차:
  1. 데이터 표준화 수행.
  2. 공분산 행렬(np.cov) 산출.
  3. 고유값(Eigenvalue) 및 고유벡터(Eigenvector) 추출 (np.linalg.eig).
  4. 고유벡터와 원본 데이터를 행렬 곱(np.dot)하여 주성분을 추출.
* 주성분의 특징:
  * 각 주성분은 서로 상관관계가 없다(비상관성).
  * 첫 번째 주성분이 데이터의 변동성(분산)을 가장 많이 설명하며, 뒤로 갈수록 표준편차(설명력)가 작아진다.


--------------------------------------------------------------------------------


## 5. 맷플롯립(Matplotlib)을 통한 데이터 시각화

시각화는 데이터의 분포와 군집 형태를 인간이 직관적으로 이해하도록 돕는 필수적인 과정이다.

* 산점도(Scatter Plot): plt.scatter()를 사용하여 두 변수 간의 관계를 좌표 평면에 점으로 표시한다.
* 시각화의 효용: 붓꽃 데이터의 경우, '꽃잎의 길이(Petal Length)'와 '너비(Petal Width)'를 축으로 산점도를 그렸을 때 종별로 확연히 구분되는 군집이 나타난다. 이는 해당 변수들이 종 분류에 중요한 피처(Feature)임을 시사한다.
* 주요 구성 요소: 그래프의 이해를 돕기 위해 제목(title), 축 이름(xlabel, ylabel), 범례(legend) 등을 설정하는 데코레이션 과정이 동반된다.

결론: 파이썬 머신러닝 분석은 판다스를 통한 데이터 정제, 넘파이를 통한 수치적 전처리 및 변환, 맷플롯립을 통한 시각적 통찰 확보의 순차적 협업 구조를 가진다. 이를 통해 복잡한 데이터셋에서 유의미한 패턴을 발견하고 모델링의 기초를 다질 수 있다.
