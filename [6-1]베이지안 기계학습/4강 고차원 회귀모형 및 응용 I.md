# 고차원 회귀모형 및 응용 I

## 1. 실행 요약 (Executive Summary)

본 문서는 선형 회귀 모형의 핵심 모수인 회귀 계수(\beta)와 오차항 분산(\sigma^2)을 추론하기 위한 베이지안 접근법을 다룬다. 특히 회귀 계수에 대한 사전 분포로 널리 사용되는 **젤너의 G-프라이어(Zellner's g-prior)**를 중점적으로 살펴본다.

주요 함의는 다음과 같다:

* 젤너의 G-프라이어의 효율성: 디자인 매트릭스(X)의 구조를 사전 분포의 공분산에 반영함으로써 계산적 편의성과 해석적 직관(사후 평균의 가중 평균 형태)을 동시에 제공한다.
* 풀 베이지안(Fully Bayesian) 접근: 초모수 g를 고정하지 않고 사전 분포(인버스 감마)를 부여하는 '젤너-시우 프라이어(Zellner-Siow prior)'를 통해 데이터가 스스로 최적의 g를 선택하게 할 수 있으며, 이는 RStan과 같은 MCMC 도구로 구현 가능하다.
* 모델의 확장성: 동일한 논리를 로지스틱 회귀 모형에도 적용할 수 있으나, 고차원 데이터(p > n) 세팅에서는 역행렬 존재 문제로 인해 젤너의 G-프라이어 사용이 불가능하며, 이를 해결하기 위한 홀슈(Horshoe) 프라이어와 같은 대안이 필요하다.

## 2. 선형 회귀 모형에서의 베이지안 추론

### 2.1 모형 설정 및 변수 정의

선형 회귀 모형은 종속 변수 Y와 p개의 공변량(독립 변수) X의 선형 결합으로 정의된다.

* 모형식: $Y = X\beta + \epsilon, \quad \epsilon \sim N(0, \sigma^2 I)$
* 추론 대상: 회귀 계수 벡터 $\beta$ (관심 대상) 및 오차항 분산 $\sigma^2$
* 행렬 표기: n개 관측치를 묶은 n차원 벡터 y와 $n \times p$ 계획 행렬(Design Matrix) X를 사용한다.

### 2.2 사전 분포(Prior)의 설정

계산적 용이성을 위해 $\beta$와 $\sigma^2$의 조인트 사전 분포를 $p(\beta, \sigma^2) = p(\beta | \sigma^2)p(\sigma^2)$ 형태로 선언한다.

| 구분 | 사용된 사전 분포 | 주요 특징 |
|:---:|:---:|:---:|
| 회귀 계수 ($\beta$) | 젤너의 G-프라이어 | 다변량 정규분포를 따르며, 공분산 행렬에 $g\sigma^2(X^T X)^{-1}$를 사용함. |
| 분산 ($\sigma^2$) | 제프리스 프라이어 | 무정보 사전 분포(Non-informative prior)의 대표격으로, $1/\sigma^2$에 비례함. |

## 3. 젤너의 G-프라이어(Zellner's g-prior) 상세 분석

### 3.1 정의 및 특징

젤너의 G-프라이어는 $\beta | \sigma^2, g \sim N(0, g\sigma^2(X^T X)^{-1})$로 정의된다.

* 평균: 일반적으로 사전 정보가 없는 경우 0 벡터를 사용한다.
* 공분산: 초모수 g, 오차 분산 $\sigma^2$, 그리고 디자인 매트릭스의 정보 ($(X^T X)^{-1}$)가 결합된 형태다. 이 구조는 사후 분포 계산을 매우 용이하게 만든다.

### 3.2 초모수 g의 선택 전략

g는 사전 분포의 영향력을 조절하는 핵심 파라미터다.

1. 고정값 사용 (g = n): 샘플 사이즈 n으로 설정하는 것이 일반적이다. 이 경우 사전 분포는 데이터 1개만큼의 정보량(Weight)을 가지는 것으로 해석되며, 사후 평균은 프라이어 평균(0)과 데이터 기반 추정치(LSE)의 가중 평균으로 나타난다.
2. 풀 베이지안 접근 (젤너-시우 프라이어): g를 확률 변수로 취급하여 $g \sim IG(1, n/2)$와 같은 인버스 감마 분포를 부여한다. 데이터가 g의 값을 결정하도록 하며, 클로즈드 폼(Closed-form) 계산이 어려워 MCMC가 필수적이다.

## 4. RStan을 이용한 사후 분포 추론 구현

MCMC 수행을 위해 RStan 패키지를 활용하며, 효율적인 계산을 위해 스탠(Stan) 코드 내 블록을 전략적으로 사용한다.

### 4.1 주요 코드 블록 구조

* Data Block: n, p, y, X 등 인풋 데이터 선언.
* Transformed Data Block: $X^T X$의 역행렬 등 MCMC 과정에서 변하지 않는 상수를 미리 계산하여 연산량을 절감한다.
* Parameters Block: 추론 대상인 $\beta, \sigma^2, g$ 선언.
* Model Block: 라이클리후드($y \sim Normal(X\beta, \sigma)$)와 사전 분포(g, $\beta, \sigma^2$) 정의.
  * 제프리스 프라이어는 target += -log(sigma_sq)와 같은 방식으로 로그 포스테리어 텀에 직접 더해준다.
* Generated Quantities Block: 사후 샘플을 이용해 새로운 데이터($X_{new}$)에 대한 예측값($y_{new}$)을 생성한다.

### 4.2 분석 결과 해석 지표

* Trace Plot: 체인이 일정한 수준에서 수렴하는지 확인.
* ACF Plot: 샘플 간 상관성이 낮아 몬테카를로 근사가 적절한지 판단.
* 95% 신용 구간(Credible Interval): 특정 계수의 구간이 0을 포함하지 않으면 해당 공변량이 유의미한 영향을 미친다고 해석한다.

## 5. 로지스틱 회귀 모형으로의 확장

### 5.1 모형의 특징

종속 변수 Y가 0 또는 1인 이진형(Binary) 자료일 때 사용한다.

* 연결 함수: P(Y=1) = $\pi$일 때, $\text{logit}(\pi) = X\beta + \alpha$ (절편 포함).
* 베이지안 설정: 선형 회귀와 달리 오차항 분산 $\sigma^2$가 없으므로 젤너의 G-프라이어에서 해당 항이 제외된다. 절편 $\alpha$에는 보통 무정보 정규 분포를 부여한다.

### 5.2 RStan 구현의 차이점

* 데이터 타입을 실수(Real)가 아닌 정수(Integer)로 선언한다.
* bernoulli_logit_glm 함수를 사용하여 효율적으로 라이클리후드를 정의한다.

## 6. 결론 및 향후 과제: 고차원 회귀 모형의 한계

젤너의 G-프라이어는 매우 강력한 도구이나, **고차원 세팅(p > n)**에서는 치명적인 한계를 가진다.

* 역행렬 문제: p가 n보다 크면 $X^T X$ 행렬이 싱귤러(Singular)해져 역행렬이 존재하지 않는다. 따라서 프라이어 정의 자체가 불가능해진다.
* 이론적 한계: 고차원에서는 일반적인 통계적 추론 방식이 정확하게 작동하지 않는 경우가 많다.
* 대안: 고차원 선형 모형의 추론을 위해서는 변수 선택이나 수축(Shrinkage) 효과가 강력한 특별한 사전 분포가 필요하며, 다음 단계로는 **홀슈 프라이어(Horseshoe Prior)**가 제시된다.
